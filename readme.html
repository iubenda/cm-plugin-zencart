Consent Manager

Consent Manager v1.0.0

Zen Cart Versions : 1.5.0 -> 1.5.8

  Description

This module allows sites to become GDPR & CCPA compliant with the Consent Manager Solution.
Further details can be accesses at https://www.consentmanager.net/.
It is recommended that you create an account at Consent Manager before installing this module.

  Installation

Pre-installation preparation
  1. Back up your database and your site
  2. Unzip the package
  3. Change the name of the 'YOUR_ADMIN' folder in the package to match the name of your admin folder. (Hint: The name of your admin folder is also the address of the admin login page).
  4. Sign in to your zen cart admin.
Uploading the files
  1. Now that you have prepared the files they are ready for upload.
  2. Sign in to your zen cart admin.
  3. Upload the renamed "YOUR_ADMIN" folder and the "includes" folder to your server.
  4. Refresh the page in zen cart admin and module will now be installed
  5. If installation is successful you will see a confirmation message at the top of the page.
  6. You will also be taken to the Consent Manager Installation Check page
Post-installation actions

There is one further action that is required before the module can be used. A single line of code needs to be inserted into your template. Why? Because it is necessary to get the Consent Manager code to load first. The code insertion is automated in most cases but if your template is unusual you will have to do the code insertion manually. It is not difficult.

  1. You should be on the Consent Manager Installation Check page
    (if not then navigate there: admin->tools->Consent Manager)
  2. In the first block you will see if the extra code has been inserted into your website.
  3. If there is a cross in this block it means that the code must still be inserted.
  4. If you see an option to 'Insert the Code' then the code can be inserted automatically.
  5. If you do not see an option to insert the code, then you will need to insert the code manually. Please read the instructions below.
  6. Now you should see a Green check mark in the first block.
    You will also see an option to rollback the changes if there have been any problems with the automatic insert.
  7. You should also see a Green check mark in the second block. If not then re-upload the module file.
  8. Finally, you should see a Dark Green check mark in the third block and the installation is complete! All you need to do now is adjust the settings.

  Use

The module has four settings

  • Enable Consent Manager: this either enables or disables the module
  • Blocking Mode: Either automatic or semiautomatic : please see the Consent Manager website for further details.
  • Enter CMP-ID: This is the CPM-ID that can be obtained from your Consent Manager account.
  • Additional HTML: allows additional HTML that Consent Manager may provide to be added.
    Tip: if the value here changes when you save it that is due to the fact that it is being encoded for secure storage in the database. There is nothing to worry about.

  Uninstallation

If you wish to uninstall the module:

  • Back up your database and your site.
  • Navigate to the Consent Manager Installation Check page
    (admin->tools->Consent Manager)
  • Click on Uninstall at the bottom of the page.
  • You will need to remove the alterations that have been made to your template manually.
    To do this edit includes/YOUR_TEMPLATE/common/html_header and remove the line below:
    Tip: you will need to change YOUR_TEMPLATE to the name of your template to find the file.
    If you are in any doubt seek help on the forum.
include(DIR_WS_MODULES . zen_get_module_directory('consentManager.php'));

  Insert Code into Your template

In rare cases, the automatic code insertion will not be able to insert the correct code into your template.
This will mean that you need to edit a file in your template.

The only change that is made to includes/YOUR_TEMPLATE/common/html_header is that one line is inserted. 

<?phpinclude(DIR_WS_MODULES . zen_get_module_directory('consentManager.php'));?>

The aim is to insert this as high up the file as is practical. Look for a line that is something like this, perhaps 

<meta name="viewport" content="width=device-width, initial-scale=1, user-scalable=yes"/>

And put it directly below that so it looks like this.

<meta name="viewport" content="width=device-width, initial-scale=1, user-scalable=yes"/>
<?php include(DIR_WS_MODULES . zen_get_module_directory('consentManager.php'));?>

It is difficut to be specific because the reason we are having to do this is that your template is unusual.
The best advice is probably from the template developer or the template thread on the Zen Cart forum.
